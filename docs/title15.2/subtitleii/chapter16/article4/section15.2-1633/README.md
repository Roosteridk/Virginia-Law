# § 15.2-1633 Part-time compensated assistants to attorneys for the Commonwealth.

<p>Notwithstanding any contrary provisions of §§ <a href='/vacode/15.2-1627.1/'>15.2-1627.1</a>, <a href='/vacode/15.2-1628/'>15.2-1628</a> and <a href='/vacode/15.2-1630/'>15.2-1630</a>, the Compensation Board at the request of the attorney for the Commonwealth may provide for one compensated part-time assistant to a full-time attorney for the Commonwealth.</p><p>1983, c. 476, § 15.1-50.2; 1994, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?941+ful+CHAP0780'>780</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?941+ful+CHAP0792'>792</a>; 1997, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?971+ful+CHAP0587'>587</a>; 2000, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?001+ful+CHAP0913'>913</a>.</p>
