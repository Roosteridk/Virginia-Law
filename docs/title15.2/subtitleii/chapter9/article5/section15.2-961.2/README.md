<p>An ordinance adopted pursuant to ยง <a href='http://law.lis.virginia.gov/vacode/15.2-961.1/'>15.2-961.1</a> may allow a locality to post signs on private property that is proposed to be redeveloped with one single-family home that notify the public that an infill lot grading plan is pending for review before the locality.
