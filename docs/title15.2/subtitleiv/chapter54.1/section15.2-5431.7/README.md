# ยง 15.2-5431.7 Filing articles of incorporation.

<p>After adoption or approval of a resolution creating an authority, the governing body of the locality shall file with the State Corporation Commission the authority's articles of incorporation.</p><p>2003, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?031+ful+CHAP0643'>643</a>.</p>
