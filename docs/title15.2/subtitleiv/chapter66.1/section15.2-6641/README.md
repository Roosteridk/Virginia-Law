# ยง 15.2-6641 Directors and persons executing bonds not liable thereon.

<p>Neither the board of directors nor any person executing the bonds shall be liable personally for the Authority's bonds by reasons of the issuance thereof.</p><p>2005, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?051+ful+CHAP0842'>842</a>.</p>
