# ยง 15.2-7415 Directors and persons executing bonds not liable thereon.

<p>Neither the board of directors nor any person executing the bonds shall be liable personally for the Authority's bonds by reasons of the issuance thereof.</p><p>2014, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?141+ful+CHAP0471'>471</a>.</p>
