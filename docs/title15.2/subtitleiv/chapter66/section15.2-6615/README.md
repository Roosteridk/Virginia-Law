# ยง 15.2-6615 Directors and persons executing bonds not liable thereon.

<p>Neither the board of directors nor any person executing the bonds shall be liable personally for the Authority's bonds by reasons of the issuance thereof.</p><p>2002, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?021+ful+CHAP0766'>766</a>.</p>
