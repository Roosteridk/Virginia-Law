# ยง 15.2-6607 Name of authority.

<p>The name of the Authority shall be the Middle Peninsula Chesapeake Bay Public Access Authority. The name of this authority may be changed upon approval of a simple majority of the directors of the Authority.</p><p>2002, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?021+ful+CHAP0766'>766</a>.</p>
