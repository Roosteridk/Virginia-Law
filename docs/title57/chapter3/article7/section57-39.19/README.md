# § 57-39.19 Application of §§ 54.1-2818.1 through 54.1-2818.3.

<p>The provisions of §§ <a href='http://law.lis.virginia.gov/vacode/54.1-2818.1/'>54.1-2818.1</a> through<span data-mce-style="font-family: &quot;PT Serif&quot;, Times, serif; font-size: 18px; font-style: normal; font-variant-caps: normal;" style="font-family: &quot;PT Serif&quot;, Times, serif; font-size: 18px; font-style: normal; font-variant-caps: normal;"> </span><a href="http://law.lis.virginia.gov/vacode/54.1-2818.3/" data-mce-style="font-family: &quot;PT Serif&quot;, Times, serif; font-size: 18px; font-style: normal; font-variant-caps: normal;" style="font-family: &quot;PT Serif&quot;, Times, serif; font-size: 18px; font-style: normal; font-variant-caps: normal;">54.1-2818.3</a> shall apply to the cremation of any dead human body.</p><p>1979, c. 724.</p>
