# § 57-38 Exemption from §§ 57-36 and 57-37.

<p>Sections <a href='http://law.lis.virginia.gov/vacode/57-36/'>57-36</a> and <a href='http://law.lis.virginia.gov/vacode/57-37/'>57-37</a> shall not apply to any graveyard or cemetery owned by a church, or controlled by trustees, in which sections are sold.</p><p>Code 1919, § 55.</p>
