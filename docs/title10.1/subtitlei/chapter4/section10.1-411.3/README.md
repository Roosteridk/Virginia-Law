# ยง 10.1-411.3 Banister State Scenic River.

<p>The Banister River from the Route 29 bridge in Pittsylvania County to the confluence with the Dan River in Halifax County, a distance of approximately 63.3 miles, is hereby designated as the Banister State Scenic River, a component of the Virginia Scenic Rivers System.</p><p>2013, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?131+ful+CHAP0352'>352</a>; 2014, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?141+ful+CHAP0149'>149</a>; 2018, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?181+ful+CHAP0273'>273</a>.</p>
