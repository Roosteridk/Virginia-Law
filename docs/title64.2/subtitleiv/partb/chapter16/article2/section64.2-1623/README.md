# § 64.2-1623 Incorporation of authority.

<p>A. An agent has authority described in this article if the power of attorney refers to general authority with respect to the descriptive term for the subjects stated in §§ <a href='http://law.lis.virginia.gov/vacode/64.2-1625/'>64.2-1625</a> through <a href='http://law.lis.virginia.gov/vacode/64.2-1638/'>64.2-1638</a>, or cites the section in which the authority is described.</p><p>B. A reference in a power of attorney to general authority with respect to the descriptive term for a subject in §§ <a href='http://law.lis.virginia.gov/vacode/64.2-1625/'>64.2-1625</a> through <a href='http://law.lis.virginia.gov/vacode/64.2-1638/'>64.2-1638</a> or a citation to a section of §§ <a href='http://law.lis.virginia.gov/vacode/64.2-1625/'>64.2-1625</a> through <a href='http://law.lis.virginia.gov/vacode/64.2-1638/'>64.2-1638</a> incorporates the entire section as if it were set out in full in the power of attorney.</p><p>C. A principal may modify authority incorporated by reference.</p><p>2010, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?101+ful+CHAP0455'>455</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?101+ful+CHAP0632'>632</a>, § 26-96; 2012, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?121+ful+CHAP0614'>614</a>.</p>
