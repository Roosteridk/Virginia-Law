# ยง 64.2-1220 Receipt for vouchers filed in settlement; effect thereof.

<p>Any commissioner of accounts having before him the accounts of a fiduciary for settlement shall, on request, execute and deliver to the fiduciary a receipt for all vouchers filed with the commissioner of accounts.
