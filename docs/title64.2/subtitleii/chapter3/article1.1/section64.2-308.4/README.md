# § 64.2-308.4 Composition of the augmented estate; marital property portion.

<p>A. Subject to § <a href='http://law.lis.virginia.gov/vacode/64.2-308.9/'>64.2-308.9</a>, the value of the augmented estate, to the extent provided in §§ <a href='http://law.lis.virginia.gov/vacode/64.2-308.5/'>64.2-308.5</a>, <a href='http://law.lis.virginia.gov/vacode/64.2-308.6/'>64.2-308.6</a>, <a href='http://law.lis.virginia.gov/vacode/64.2-308.7/'>64.2-308.7</a>, and <a href='http://law.lis.virginia.gov/vacode/64.2-308.8/'>64.2-308.8</a>, consists of the sum of the values of all property, whether real or personal, movable or immovable, tangible or intangible, wherever situated, that constitute:</p><p>1. The decedent's net probate estate;</p><p>2. The decedent's non-probate transfers to others;</p><p>3. The decedent's non-probate transfers to the surviving spouse; and</p><p>4.
