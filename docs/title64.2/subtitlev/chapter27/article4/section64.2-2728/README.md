# ยง 64.2-2728 Authority to release.

<p>A powerholder may release a power of appointment, in whole or in part, except to the extent that the terms of the instrument creating the power prevent the release.</p><p>2016, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?161+ful+CHAP0266'>266</a>.</p>
