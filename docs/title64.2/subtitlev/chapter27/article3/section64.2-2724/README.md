# ยง 64.2-2724 Disposition of unappointed property if partial appointment to taker in default.

<p>Unless the terms of the instrument creating or exercising a power of appointment manifest a contrary intent, if the powerholder makes a valid partial appointment to a taker in default of appointment, the taker in default of appointment may share fully in unappointed property.</p><p>2016, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?161+ful+CHAP0266'>266</a>.</p>
