# ยง 64.2-2208 Effective date.

<p>An act done before July 1, 1994, in any proceeding and any accrued right is not impaired by this chapter. If a right is acquired, extinguished, or barred upon the expiration of a prescribed period of time that has commenced to run by the provisions of any statute before July 1, 1994, the provisions remain in force with respect to that right.</p><p>Any rule of construction or presumption provided in this chapter applies to instruments executed and multiple-party accounts opened before July 1, 1994, unless there is a clear indication of a contrary intent.</p><p>1994, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?941+ful+CHAP0475'>475</a>, ยง 64.1-104.9; 2012, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?121+ful+CHAP0614'>614</a>.</p>
