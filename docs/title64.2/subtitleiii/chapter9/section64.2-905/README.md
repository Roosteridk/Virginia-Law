# § 64.2-905 Multiple beneficiaries; separate custodial trusts; survivorship.

<p>A. Beneficial interests in a custodial trust created for multiple beneficiaries are deemed to be separate custodial trusts of equal undivided interests for each beneficiary. Except in a transfer or declaration for use and benefit of spouses, for whom survivorship is presumed, a right of survivorship does not exist unless the instrument creating the custodial trust specifically provides for survivorship or survivorship is required as to marital property.</p><p>B. Custodial trust property held under this chapter by the same custodial trustee for the use and benefit of the same beneficiary may be administered as a single custodial trust.</p><p>C. A custodial trustee of custodial trust property held for more than one beneficiary shall separately account to each beneficiary pursuant to §§ <a href='/vacode/64.2-906/'>64.2-906</a> and <a href='/vacode/64.2-914/'>64.2-914</a> for the administration of the custodial trust.</p><p>1990, c. 264, § 55-34.6; 2012, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?121+ful+CHAP0614'>614</a>; 2020, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?201+ful+CHAP0900'>900</a>.</p>
