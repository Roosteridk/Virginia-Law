# ยง 64.2-779.24 Obligations.

<p>A debt, liability, or other obligation enforceable against property of a first trust is enforceable to the same extent against the property when held by the second trust after exercise of the decanting power.</p><p>2017, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?171+ful+CHAP0592'>592</a>.</p>
