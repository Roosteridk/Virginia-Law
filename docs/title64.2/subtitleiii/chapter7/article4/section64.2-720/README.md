# ยง 64.2-720 Requirements for creation.

<p>A. A trust is created only if:</p><p>1. The settlor has capacity to create a trust; or when the trust is created by the settlor's agent under a power of attorney, which expressly authorizes the agent to create a trust on the settlor's behalf;</p><p>2. The settlor or his agent indicates an intention to create the trust;</p><p>3. The trust has a definite beneficiary or is:</p><p>a. A charitable trust;</p><p>b. A trust for the care of an animal, as provided in ยง <a href='http://law.lis.virginia.gov/vacode/64.2-726/'>64.2-726</a>; or</p><p>c. A trust for a noncharitable purpose, as provided in ยง <a href='http://law.lis.virginia.gov/vacode/64.2-727/'>64.2-727</a>;</p><p>4. The trustee has duties to perform; and</p><p>5. The same person is not the sole trustee and sole beneficiary.</p><p>B. A beneficiary is definite if the beneficiary can be ascertained now or in the future, subject to any applicable rule against perpetuities.</p><p>C. A power in a trustee to select a beneficiary from an indefinite class is valid. If the power is not exercised within a reasonable time, the power fails and the property subject to the power passes to the persons who would have taken the property had the power not been conferred.</p><p>2005, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?051+ful+CHAP0935'>935</a>, ยง 55-544.02; 2010, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?101+ful+CHAP0455'>455</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?101+ful+CHAP0632'>632</a>; 2012, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?121+ful+CHAP0614'>614</a>.</p>
