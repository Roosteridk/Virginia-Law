# ยง 64.2-798 Event affecting administration or distribution.

<p>If the happening of an event, including marriage, divorce, performance of educational requirements, or death, affects the administration or distribution of a trust, a trustee who has exercised reasonable care to ascertain the happening of the event is not liable for a loss resulting from the trustee's lack of knowledge.</p><p>2005, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?051+ful+CHAP0935'>935</a>, ยง 55-550.07; 2012, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?121+ful+CHAP0614'>614</a>.</p>
