# ยง 38.2-3569 Funding of external review.

<p>The health carrier against which a request for an external review is filed shall pay the cost incurred by the independent review organization in conducting the external review.</p><p>2011, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?111+ful+CHAP0788'>788</a>.</p>
