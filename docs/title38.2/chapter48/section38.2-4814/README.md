# § 38.2-4814 Penalties.

<p>Any violation of this chapter shall be punished as provided for in §§ <a href='http://law.lis.virginia.gov/vacode/38.2-218/'>38.2-218</a> and <a href='http://law.lis.virginia.gov/vacode/38.2-1831/'>38.2-1831</a>.</p><p>Code 1950, § 38.1-314.15; 1960, c. 503; 1979, c. 513, § 38.1-327.59; 1986, c. 562.</p>
