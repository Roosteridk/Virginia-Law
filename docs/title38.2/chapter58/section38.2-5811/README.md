# ยง 38.2-5811 Controversies involving contracts.

<p>The Commission shall have no jurisdiction to adjudicate controversies between an MCHIP and its covered persons, and a breach of contract shall not be deemed a violation of this chapter.</p><p>1998, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?981+ful+CHAP0891'>891</a>.</p>
