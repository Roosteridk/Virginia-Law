# ยง 38.2-5013 Limitation on claims.

<p>Any claim under this chapter that is filed more than ten years after the birth of an infant alleged to have a birth-related neurological injury is barred.</p><p>1987, c. 540; 1999, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?991+ful+CHAP0806'>806</a>; 2006, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?061+ful+CHAP0919'>919</a>.</p>
