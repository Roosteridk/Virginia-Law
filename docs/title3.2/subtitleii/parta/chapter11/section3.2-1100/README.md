# § 3.2-1100 Diversion of dedicated revenues.

<p>A. The unexpended balances of the following special funds shall not be diverted or expended for any purpose other than each fund's intended purpose. The special funds are:</p><p>1. Apple Fund (§ <a href='/vacode/3.2-1206/'>3.2-1206</a>);</p><p>2. Peanut Fund (§ <a href='/vacode/3.2-1906/'>3.2-1906</a>);</p><p>3. Plant Pollination Fund (§ <a href='/vacode/3.2-2806/'>3.2-2806</a>);</p><p>4. Virginia Agricultural Foundation Fund (§ <a href='/vacode/3.2-2905/'>3.2-2905</a>);</p><p>5. Virginia Bright Flue-Cured Tobacco Promotion Fund (§ <a href='/vacode/3.2-2407/'>3.2-2407</a>);</p><p>6. Virginia Cattle Industry Fund (§ <a href='/vacode/3.2-1305/'>3.2-1305</a>);</p><p>7. Virginia Corn Fund (§ <a href='/vacode/3.2-1411/'>3.2-1411</a>);</p><p>8. Virginia Cotton Fund (§ <a href='/vacode/3.2-1511/'>3.2-1511</a>);</p><p>9. Virginia Dark-Fired Tobacco Promotion Fund (§ <a href='/vacode/3.2-2407/'>3.2-2407</a>.1);</p><p>10. Virginia Egg Fund (§ <a href='/vacode/3.2-1605/'>3.2-1605</a>);</p><p>11. Virginia Horse Industry Promotion and Development Fund (§ <a href='/vacode/3.2-1704/'>3.2-1704</a>);</p><p>12. Virginia Marine Products Fund (§ <a href='/vacode/3.2-2705/'>3.2-2705</a>);</p><p>13. Virginia Milk Commission Assessments Fund (§ <a href='/vacode/3.2-3220/'>3.2-3220</a>);</p><p>14. Virginia Pork Industry Fund (§ <a href='/vacode/3.2-2005/'>3.2-2005</a>);</p><p>15. Virginia Potato Fund (§ <a href='/vacode/3.2-1810/'>3.2-1810</a>);</p><p>16. Virginia Sheep Industry Promotion and Development Fund (§ <a href='/vacode/3.2-2111/'>3.2-2111</a>);</p><p>17. Virginia Small Grains Fund (§ <a href='/vacode/3.2-2211/'>3.2-2211</a>);</p><p>18. Virginia Soybean Fund (§ <a href='/vacode/3.2-2311/'>3.2-2311</a>); and</p><p>19. Virginia Wine Promotion Fund (§ <a href='/vacode/3.2-3005/'>3.2-3005</a>).</p><p>B.
