# ยง 20-181 Confidentiality of collaborative law communication.

<p>A collaborative law communication is confidential to the extent agreed upon by the parties in a signed record or as provided by another law of the Commonwealth.</p><p>2021, Sp. Sess. I, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?212+ful+CHAP0346'>346</a>.</p>
