# ยง 20-186 Uniformity of application and construction.

<p>In applying and construing this uniform chapter, consideration shall be given to the need to promote uniformity of the law with respect to its subject matter among states that enact it.</p><p>2021, Sp. Sess. I, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?212+ful+CHAP0346'>346</a>.</p>
