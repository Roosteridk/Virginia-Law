<p>A party or support enforcement agency seeking to modify, or to modify and enforce, a child support order issued in another state shall register that order in this Commonwealth in the same manner as provided in Article 8 (ยง <a href='http://law.lis.virginia.gov/vacode/20-88.66/'>20-88.66</a> et seq.) if the order has not been registered.
