# ยง 20-88.68 Effect of registration for enforcement.

<p>A. A support order or income-withholding order issued in another state or a foreign support order is registered when the order is filed in the registering tribunal of the Commonwealth.</p><p>B. A registered support order issued in another state or in a foreign country is enforceable in the same manner and is subject to the same procedures as an order issued by a tribunal of the Commonwealth.</p><p>C. Except as otherwise provided in this chapter, a tribunal of the Commonwealth shall recognize and enforce, but may not modify, a registered order if the issuing tribunal had jurisdiction.</p><p>1994, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?941+ful+CHAP0673'>673</a>; 2015, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?151+ful+CHAP0727'>727</a>.</p>
