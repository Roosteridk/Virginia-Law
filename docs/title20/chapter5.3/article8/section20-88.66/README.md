# ยง 20-88.66 Registration of order for enforcement.

<p>A foreign support order, or a support order or an income-withholding order issued in another state, may be registered in the Commonwealth for enforcement.</p><p>1994, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?941+ful+CHAP0673'>673</a>; 2015, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?151+ful+CHAP0727'>727</a>.</p>
