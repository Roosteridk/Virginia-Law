# ยง 20-146.37 Transitional provision.

<p>A motion or other request for relief made in a child custody proceeding or to enforce a child custody determination that was commenced before July 1, 2001, is governed by the law in effect at the time the motion or other request was made.</p><p>2001, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?011+ful+CHAP0305'>305</a>.</p>
