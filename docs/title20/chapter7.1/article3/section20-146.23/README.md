# ยง 20-146.23 Enforcement under Hague Convention.

<p>Under this article a court of this Commonwealth may enforce an order for the return of the child made under the Hague Convention on the Civil Aspects of International Child Abduction.</p><p>2001, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?011+ful+CHAP0305'>305</a>.</p>
