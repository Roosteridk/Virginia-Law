# ยง 8.01-216.19 Application of the Rules of the Supreme Court.

<p>The Rules of the Supreme Court of Virginia shall apply to all proceedings under this article, except when those Rules are inconsistent with this article.</p><p>2002, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?021+ful+CHAP0842'>842</a>.</p>
