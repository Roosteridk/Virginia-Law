# ยง 8.01-267.2 When actions pending in same court.

<p>For purposes of this chapter, actions shall be considered pending in the same circuit court when they have been (i) filed in that court, regardless of whether the defendant has been served with process, or (ii) properly transferred to that court.</p><p>1995, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?951+ful+CHAP0555'>555</a>.</p>
