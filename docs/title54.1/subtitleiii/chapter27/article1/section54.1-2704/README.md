# ยง 54.1-2704 Nominations.

<p>Nominations may be made for each professional vacancy from a list of three names submitted to the Governor by the Virginia Dental Association, the Old Dominion State Dental Society, the Virginia Dental Hygienists' Association, and the Commonwealth Dental Hygienists' Society. Further, any licensee of this chapter may submit nominations to the Governor.
