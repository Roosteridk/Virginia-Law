# ยง 54.1-2910.3 No requirement to participate in third-party reimbursement programs.

<p>No provider licensed pursuant to this chapter shall be required to participate in any public or private third-party reimbursement program as a condition of licensure.</p><p>2011, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?111+ful+CHAP0490'>490</a>.</p>
