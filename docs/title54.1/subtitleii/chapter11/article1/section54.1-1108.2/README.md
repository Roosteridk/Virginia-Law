# ยง 54.1-1108.2 Application for Class C license; fees; issuance.

<p>A. Any person desiring to be licensed as a Class C contractor shall file with the Department a written application on a form prescribed by the Board. The application shall be accompanied by a fee set by the Board pursuant to ยง <a href='/vacode/54.1-201/'>54.1-201</a>. The application shall contain information concerning the name, location, nature, and operation of the business, and information demonstrating that the applicant possesses the character and minimum skills to properly engage in the occupation of contracting.</p><p>B. The Board may grant a Class C license in any of the following classifications: (i) residential building contractor, (ii) commercial building contractor, (iii) highway/heavy contractor, (iv) electrical contractor, (v) plumbing contractor, (vi) heating, ventilation, and air conditioning contractor, (vii) fire sprinkler contractor, and (viii) specialty contractor.</p><p>1995, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?951+ful+CHAP0771'>771</a>; 1997, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?971+ful+CHAP0885'>885</a>; 1998, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?981+ful+CHAP0754'>754</a>; 2003, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?031+ful+CHAP0892'>892</a>; 2013, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?131+ful+CHAP0116'>116</a>; 2019, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?191+ful+CHAP0726'>726</a>.</p>
