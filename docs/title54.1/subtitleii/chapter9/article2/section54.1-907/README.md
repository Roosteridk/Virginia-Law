# § 54.1-907 Fee for original license and license renewal.

<p>Upon the application for license as a branch pilot and each renewal thereof, the applicant for license or license renewal shall pay a fee established by the Board pursuant to § <a href='http://law.lis.virginia.gov/vacode/54.1-113/'>54.1-113</a>.</p><p>Code 1950, §§ 54-539, 54-540; 1988, c. 765; 1990, c. 72.</p>
