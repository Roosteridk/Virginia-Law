# ยง 54.1-3913 State Bar Fund; receipts; disbursements.

<p>The State Bar Fund is continued as a special fund in the state treasury. All fees collected from the members of the Virginia State Bar as provided in ยง <a href='http://law.lis.virginia.gov/vacode/54.1-3912/'>54.1-3912</a> shall be paid into the state treasury immediately upon collection and credited to the State Bar Fund.
