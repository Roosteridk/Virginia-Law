# ยง 33.2-911 Permissible uses by counties of certain discontinued secondary highways.

<p>Whenever a secondary highway is discontinued under ยง <a href='http://law.lis.virginia.gov/vacode/33.2-908/'>33.2-908</a>, the highway shall continue to be available for use by the public as a highway, unless it has been abandoned pursuant to this chapter or its use has been modified by an ordinance adopted pursuant to this section.
