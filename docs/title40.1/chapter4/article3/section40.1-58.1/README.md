# ยง 40.1-58.1 Application of article to public employers and employees.

<p>As used in this article, the words, "person," "persons," "employer," "employees," "union," "labor union," "association," "organization" and "corporation" shall include but not be limited to public employers, public employees and any representative of public employees in this Commonwealth.
