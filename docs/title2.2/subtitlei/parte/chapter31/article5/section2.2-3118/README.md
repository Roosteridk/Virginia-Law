# § 2.2-3118 Disclosure form; certain citizen members.

<p>The financial disclosure form to be used for filings required pursuant to subsection B of <a href='http://law.lis.virginia.gov/vacode/2.2-3114/'>§ 2.2-3114</a> and subsection B of <a href='http://law.lis.virginia.gov/vacode/2.2-3115/'>§ 2.2-3115</a> shall be filed in accordance with the provisions of <a href='http://law.lis.virginia.gov/vacode/30-356/'>§ 30-356</a>. The financial disclosure form shall be prescribed by the Council. Except as otherwise provided in <a href='http://law.lis.virginia.gov/vacode/2.2-3115/'>§ 2.2-3115</a>, all completed forms shall be filed electronically with the Council in accordance with the standards approved by it pursuant to <a href='http://law.lis.virginia.gov/vacode/30-356/'>§ 30-356</a>.</p><p>1988, c. 849, § 2.1-639.15:1; 1996, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?961+ful+CHAP0077'>77</a>; 2001, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?011+ful+CHAP0844'>844</a>; 2006, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?061+ful+CHAP0779'>779</a>; 2011, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?111+ful+CHAP0123'>123</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?111+ful+CHAP0177'>177</a>; 2014, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?141+ful+CHAP0792'>792</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?141+ful+CHAP0804'>804</a>; 2015, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?151+ful+CHAP0763'>763</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?151+ful+CHAP0777'>777</a>; 2016, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?161+ful+CHAP0773'>773</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?161+ful+CHAP0774'>774</a>.</p>
