# § 2.2-3131 Exemptions.

<p>A. The requirements of § <a href='http://law.lis.virginia.gov/vacode/2.2-3130/'>2.2-3130</a> shall not apply to state filers with a state agency who have taken an equivalent ethics orientation course through another state agency within the time periods set forth in subdivision 1 or 2 of § <a href='http://law.lis.virginia.gov/vacode/2.2-3130/'>2.2-3130</a>, as applicable.</p><p>B. State agencies may jointly conduct and state filers from more than one state agency may jointly attend an orientation course required by § <a href='http://law.lis.virginia.gov/vacode/2.2-3128/'>2.2-3128</a>, as long as the course content is relevant to the official duties of the attending state filers.</p><p>C. Before conducting each orientation course required by § <a href='http://law.lis.virginia.gov/vacode/2.2-3128/'>2.2-3128</a>, state agencies shall consult with the Attorney General and the Virginia Conflict of Interest and Ethics Advisory Council regarding appropriate course content.</p><p>2004, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?041+ful+CHAP0134'>134</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?041+ful+CHAP0392'>392</a>; 2014, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?141+ful+CHAP0792'>792</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?141+ful+CHAP0804'>804</a>.</p>
