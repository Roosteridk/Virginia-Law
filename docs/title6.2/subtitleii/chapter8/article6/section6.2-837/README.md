# ยง 6.2-837 Interstate branching by Virginia state banks.

<p>A. With the prior approval of the Commission, any Virginia state bank may establish and maintain a de novo branch or acquire a branch in a state other than the Commonwealth.</p><p>B. A Virginia state bank desiring to establish and maintain a branch in another state under this section shall file an application on a form prescribed by the Commission and pay the branch application fee set forth in ยง <a href='http://law.lis.virginia.gov/vacode/6.2-908/'>6.2-908</a>. If the Commission finds that the applicant has the financial resources sufficient to undertake the proposed expansion without adversely affecting its soundness and that the laws of the host state permit the establishment of the branch, it may approve the application. The bank may establish the branch when it has received the written approval of the Commission.</p><p>1995, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?951+ful+CHAP0301'>301</a>, ยง 6.1-44.3; 2010, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?101+ful+CHAP0794'>794</a>; 2014, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?141+ful+CHAP0200'>200</a>.</p>
