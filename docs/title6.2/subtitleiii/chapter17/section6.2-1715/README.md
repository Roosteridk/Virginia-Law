# ยง 6.2-1715 Advertising; use of a unique identifier.

<p>A. No individual required to be licensed under this chapter shall use or cause to be published any advertisement that:</p><p>1. Contains any false, misleading, or deceptive statement or representation; or</p><p>2. Except as otherwise provided by the Commission, identifies a licensee by any name other than the name set forth on the license issued by the Commission.</p><p>B.
