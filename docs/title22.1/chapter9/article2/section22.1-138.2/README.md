# ยง 22.1-138.2 Certain public school buildings; carbon monoxide detectors.

<p>Each public school building that was built before 2015 and that houses any classroom for students shall be equipped with at least one carbon monoxide detector.</p><p>2021, Sp. Sess. I, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?212+ful+CHAP0165'>165</a>.</p>
