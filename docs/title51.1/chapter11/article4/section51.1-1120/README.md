# ยง 51.1-1120 Initial benefit period.

<p>During the first seven calendar days following the commencement of a disability, a participating employee may use sick leave, family and personal leave, and annual and compensatory leave to provide income replacement.</p><p>1998, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?981+ful+CHAP0774'>774</a>.</p>
