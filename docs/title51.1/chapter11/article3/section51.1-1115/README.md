# ยง 51.1-1115 Rehabilitation incentive.

<p>Disability benefits payable to a participating employee who fails to cooperate with a rehabilitation program prescribed for the employee shall be decreased by fifty percent of the amounts otherwise payable to such employee.</p><p>1998, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?981+ful+CHAP0774'>774</a>.</p>
