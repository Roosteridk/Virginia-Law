# § 5.1-2.24 Applicability of certain sections.

<p>The provisions of §§ <a href='http://law.lis.virginia.gov/vacode/5.1-2.5/'>5.1-2.5</a>, <a href='http://law.lis.virginia.gov/vacode/5.1-2.7/'>5.1-2.7</a>, <a href='http://law.lis.virginia.gov/vacode/5.1-2.8/'>5.1-2.8</a>, <a href='http://law.lis.virginia.gov/vacode/5.1-2.9/'>5.1-2.9</a>, <a href='http://law.lis.virginia.gov/vacode/5.1-2.10/'>5.1-2.10</a>, <a href='http://law.lis.virginia.gov/vacode/5.1-2.11/'>5.1-2.11</a>, <a href='http://law.lis.virginia.gov/vacode/5.1-2.12/'>5.1-2.12</a>, <a href='http://law.lis.virginia.gov/vacode/5.1-2.13/'>5.1-2.13</a>, and <a href='http://law.lis.virginia.gov/vacode/5.1-2.20/'>5.1-2.20</a> of this chapter shall be applicable only to any airport owned by the Virginia Airports Authority or its successor agency on January 1, 1980.</p><p>1980, c. 750.</p>
