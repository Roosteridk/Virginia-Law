# ยง 46.2-2071 Required permit.

<p>No nonprofit/tax-exempt passenger carrier, unless otherwise exempted, shall transport passengers on any highway within the Commonwealth on an intrastate basis without first having obtained from the Department a permit authorizing such operation.</p><p>2001, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?011+ful+CHAP0596'>596</a>.</p>
