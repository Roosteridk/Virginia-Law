# ยง 46.2-2075 Required certificates of public convenience and necessity.

<p>No common carrier not otherwise exempted, other than a sight-seeing carrier, shall engage in intrastate operation on any highway within the Commonwealth without first having obtained from the Department a certificate of public convenience and necessity authorizing such operation.</p><p>2001, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?011+ful+CHAP0596'>596</a>; 2011, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?111+ful+CHAP0881'>881</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?111+ful+CHAP0889'>889</a>.</p>
