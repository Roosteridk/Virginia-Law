# ยง 46.2-2083 Schedule title page and content.

<p>A. Title page of time schedules shall contain the following:</p><p>1. Time schedules must be numbered consecutively in the upper right hand corner, beginning with No. 1, and show the number of the time schedule cancelled thereby, if any.</p><p>2. Name of the motor carrier.</p><p>3. The termini or points between which the time schedules apply.</p><p>4. Date issued and date effective.</p><p>5. The name, title, and address of the officer issuing such time schedule, including street address.</p><p>B. Time schedules shall show:</p><p>1. The time of departure from all termini.</p><p>2. The time of departure from intermediate points between termini.</p><p>3. What points, if any, on route of carrier at which service cannot be rendered.</p><p>2001, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?011+ful+CHAP0596'>596</a>.</p>
