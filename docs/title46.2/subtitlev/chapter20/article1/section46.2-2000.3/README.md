# ยง 46.2-2000.3 Disposition of funds collected.

<p>Except as otherwise provided, all fees collected by the Department pursuant to this chapter shall be paid into the state treasury and set aside as a special fund to be used to meet the expenses of the Department.</p><p>2001, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?011+ful+CHAP0596'>596</a>.</p>
