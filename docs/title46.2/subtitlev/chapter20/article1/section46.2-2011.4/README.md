# ยง 46.2-2011.4 Conversion of contract bus certificates.

<p>All contract bus carriers that hold a certificate issued prior to July 1, 2012, shall be issued a replacement certificate of fitness as a contract passenger carrier. The holder of such certificate shall not be required to apply for a replacement certificate.</p><p>2001, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?011+ful+CHAP0596'>596</a>; 2002, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?021+ful+CHAP0861'>861</a>; 2011, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?111+ful+CHAP0881'>881</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?111+ful+CHAP0889'>889</a>; 2012, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?121+ful+CHAP0022'>22</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?121+ful+CHAP0111'>111</a>.</p>
