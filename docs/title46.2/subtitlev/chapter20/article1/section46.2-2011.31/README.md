# ยง 46.2-2011.31 Licenses, taxes, etc., not affected.

<p>Nothing in this chapter shall be construed to relieve any person from the payment of any licenses, fees, taxes or levies now or hereafter imposed by law.</p><p>2001, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?011+ful+CHAP0596'>596</a>.</p>
