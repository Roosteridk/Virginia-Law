# ยง 46.2-2164 Constructive weight.

<p>If no certified scale is available at origin, at any point enroute, or at destination, a constructive weight based upon seven pounds per cubic foot of properly loaded van space may be used.</p><p>2001, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?011+ful+CHAP0596'>596</a>.</p>
