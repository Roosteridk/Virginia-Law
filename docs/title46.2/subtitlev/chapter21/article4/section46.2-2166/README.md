# ยง 46.2-2166 Minimum weight shipments, notice.

<p>No carrier shall accept an order for a shipment for transportation that appears to be subject to the minimum weight provisions of the carrier's tariff without first having advised the shipper of such minimum weight provisions.</p><p>2001, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?011+ful+CHAP0596'>596</a>.</p>
