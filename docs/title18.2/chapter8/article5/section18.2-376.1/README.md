# § 18.2-376.1 Enhanced penalties for using a computer in certain violations.

<p>Any person who uses a computer in connection with a violation of §§ <a href='http://law.lis.virginia.gov/vacode/18.2-374/'>18.2-374</a>, <a href='http://law.lis.virginia.gov/vacode/18.2-375/'>18.2-375</a>, or § <a href='http://law.lis.virginia.gov/vacode/18.2-376/'>18.2-376</a> is guilty of a separate and distinct Class 1 misdemeanor, and for a second or subsequent such offense within 10 years of a prior such offense is guilty of a Class 6 felony, the penalties to be imposed in addition to any other punishment otherwise prescribed for a violation of any of those sections.</p><p>2003, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?031+ful+CHAP0987'>987</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?031+ful+CHAP1016'>1016</a>.</p>
