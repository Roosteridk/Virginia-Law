# ยง 18.2-152.3 Computer fraud; penalty.

<p>Any person who uses a computer or computer network, without authority and:</p><p>1. Obtains property or services by false pretenses;</p><p>2. Embezzles or commits larceny; or</p><p>3. Converts the property of another;</p><p>is guilty of the crime of computer fraud.</p><p>If the value of the property or services obtained is $1,000 or more, the crime of computer fraud shall be punishable as a Class 5 felony. Where the value of the property or services obtained is less than $1,000, the crime of computer fraud shall be punishable as a Class 1 misdemeanor.</p><p>1984, c. 751; 1985, c. 322; 2003, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?031+ful+CHAP0987'>987</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?031+ful+CHAP1016'>1016</a>; 2005, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?051+ful+CHAP0747'>747</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?051+ful+CHAP0761'>761</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?051+ful+CHAP0827'>827</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?051+ful+CHAP0837'>837</a>; 2018, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?181+ful+CHAP0764'>764</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?181+ful+CHAP0765'>765</a>; 2020, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?201+ful+CHAP0089'>89</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?201+ful+CHAP0401'>401</a>.</p>
