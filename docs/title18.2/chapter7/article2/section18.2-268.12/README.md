# § 18.2-268.12 Ordinances.

<p>The governing bodies of counties, cities and towns are authorized to adopt ordinances paralleling the provisions of §§ <a href='http://law.lis.virginia.gov/vacode/18.2-268.1/'>18.2-268.1</a> through <a href='http://law.lis.virginia.gov/vacode/18.2-268.11/'>18.2-268.11</a>.</p><p>1992, c. 830.</p>
