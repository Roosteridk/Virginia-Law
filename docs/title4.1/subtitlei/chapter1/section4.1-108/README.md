# ยง 4.1-108 Hearings; representation by counsel.

<p>Any licensee or applicant for any license granted by the Board shall have the right to be represented by counsel at any Board hearing for which he has received notice. The licensee or applicant shall not be required to be represented by counsel during such hearing.
