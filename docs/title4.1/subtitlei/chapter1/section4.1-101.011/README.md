# ยง 4.1-101.011 Reversion to the Commonwealth.

<p>In the event of the dissolution of the Authority, all assets of the Authority, after satisfaction of creditors, shall revert to the Commonwealth.</p><p>2015, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?151+ful+CHAP0038'>38</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?151+ful+CHAP0730'>730</a>.</p>
