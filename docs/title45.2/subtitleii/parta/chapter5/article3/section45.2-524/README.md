# § 45.2-524  Examination fees.

<p>A reasonable fee in an amount set by the Board of Coal Mining Examiners, not to exceed $50, shall be paid to the Chief by each person examined before the commencement of the examination. Fees collected shall be deposited in the Coal Mining Examiners' Fund created by § <a href='/vacode/45.2-523/'>45.2-523</a>.</p><p>Code 1950, §§ 45-24, 45-25; 1954, c. 191; 1966, c. 594, §§ 45.1-8, 45.1-9; 1972, c. 440; 1978, c. 120; 1984, c. 590; 1994, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?941+ful+CHAP0028'>28</a>, § 45.1-161.31; 1996, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?961+ful+CHAP0774'>774</a>; 2007, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?071+ful+CHAP0894'>894</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?071+ful+CHAP0914'>914</a>; 2021, Sp. Sess. I, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?212+ful+CHAP0387'>387</a>.</p>
