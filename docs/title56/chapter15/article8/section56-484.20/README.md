# ยง 56-484.20 Charges for emergency calls.

<p>The MLTS provider of any multiline telephone system shall maintain and operate the MLTS in such manner that an individual placing an emergency call from a telephone station on the MLTS is not charged for the call.</p><p>2007, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?071+ful+CHAP0427'>427</a>.</p>
