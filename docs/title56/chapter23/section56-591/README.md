# ยง 56-591 Application of antitrust laws.

<p>Nothing in this chapter shall be construed to exempt or immunize from punishment or prosecution, conduct violative of federal antitrust laws, or the antitrust laws of this Commonwealth.</p><p>1999, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?991+ful+CHAP0411'>411</a>.</p>
