# ยง 56-249.7 Certain directors and officers of utility to file shareholder information.

<p>The directors and officers of any public utility as defined in ยง <a href='http://law.lis.virginia.gov/vacode/56-232/'>56-232</a> shall file with the Commission a record of all officers and directorships and all sources of income in excess of $25,000 per year arising from voting securities in all other corporations which to the knowledge of the director or officer furnishes fuel with a value in excess of $50,000 per year to the public utility.
