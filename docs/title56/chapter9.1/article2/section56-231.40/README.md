# ยง 56-231.40 Names.

<p>The words "electric cooperative" or "utility aggregation cooperative" shall not be used in the corporate name of any corporation other than those subject to this chapter, or their wholly owned subsidiaries.</p><p>1999, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?991+ful+CHAP0874'>874</a>; 2000, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?001+ful+CHAP0944'>944</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?001+ful+CHAP0999'>999</a>.</p>
