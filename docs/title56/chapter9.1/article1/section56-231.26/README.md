# ยง 56-231.26 Covenants in connection with obligations.

<p>In connection with the issuance of any obligations a cooperative may make covenants or agreements and do any and all acts or things that a corporation can make or do under the laws of this Commonwealth.</p><p>1999, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?991+ful+CHAP0874'>874</a>.</p>
