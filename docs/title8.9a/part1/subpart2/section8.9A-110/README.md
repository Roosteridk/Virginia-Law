# § 8.9A-110 Security interests arising under Title 8.2 or Title 8.2A.

<p>A security interest arising under §§ <a href='http://law.lis.virginia.gov/vacode/8.2-401/'>8.2-401</a>, <a href='http://law.lis.virginia.gov/vacode/8.2-505/'>8.2-505</a>, <a href='http://law.lis.virginia.gov/vacode/8.2-711/'>8.2-711</a> (3), or § <a href='http://law.lis.virginia.gov/vacode/8.2A-508/'>8.2A-508</a> (5) is subject to this title. However, until the debtor obtains possession of the goods:</p><p>(1) the security interest is enforceable, even if subdivision (b) (3) of § <a href='http://law.lis.virginia.gov/vacode/8.9A-203/'>8.9A-203</a> has not been satisfied;</p><p>(2) filing is not required to perfect the security interest;</p><p>(3) the rights of the secured party after default by the debtor are governed by Title 8.2 or Title 8.2A; and</p><p>(4) the security interest has priority over a conflicting security interest created by the debtor.</p><p>1964, c. 219, § 8.9-113; 1991, c. 536; 2000, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?001+ful+CHAP1007'>1007</a>.</p>
