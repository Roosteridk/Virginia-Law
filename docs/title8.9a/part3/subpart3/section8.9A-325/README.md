# § 8.9A-325 Priority of security interests in transferred collateral.

<p>(a) Subordination of security interest in transferred collateral. Except as otherwise provided in subsection (b), a security interest created by a debtor is subordinate to a security interest in the same collateral created by another person if:</p><p>(1) the debtor acquired the collateral subject to the security interest created by the other person;</p><p>(2) the security interest created by the other person was perfected when the debtor acquired the collateral; and</p><p>(3) there is no period thereafter when the security interest is unperfected.</p><p>(b) Limitation of subsection (a) subordination. Subsection (a) subordinates a security interest only if the security interest:</p><p>(1) otherwise would have priority solely under § <a href='http://law.lis.virginia.gov/vacode/8.9A-322/'>8.9A-322</a> (a) or § <a href='http://law.lis.virginia.gov/vacode/8.9A-324/'>8.9A-324</a>; or</p><p>(2) arose solely under § <a href='http://law.lis.virginia.gov/vacode/8.2-711/'>8.2-711</a> (3) or § <a href='http://law.lis.virginia.gov/vacode/8.2A-508/'>8.2A-508</a> (5).</p><p>2000, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?001+ful+CHAP1007'>1007</a>.</p>
