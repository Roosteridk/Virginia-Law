# ยง 8.9A-606 Time of default for agricultural lien.

<p>For purposes of this part, a default occurs in connection with an agricultural lien at the time the secured party becomes entitled to enforce the lien in accordance with the statute under which it was created.</p><p>2000, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?001+ful+CHAP1007'>1007</a>.</p>
