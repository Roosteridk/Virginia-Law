# ยง 8.9A-612 Timeliness of notification before disposition of collateral.

<p>A notification of disposition sent after default and ten days or more before the earliest time of disposition set forth in the notification is considered sent within a reasonable time before the disposition.</p><p>2000, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?001+ful+CHAP1007'>1007</a>.</p>
