# ยง 59.1-502.7 Formation; releases of informational rights.

<p>(a) A release is effective without consideration if it is:</p><p>(1) in a record to which the releasing party agrees, such as by manifesting assent, and which identifies the informational rights released; or</p><p>(2) enforceable under estoppel, implied license, or other law.</p><p>(b) A release continues for the duration of the informational rights released if the release does not specify its duration and does not require affirmative performance after the grant of the release by:</p><p>(1) the party granting the release; or</p><p>(2) the party receiving the release, except for relatively insignificant acts.</p><p>2000, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?001+ful+CHAP0101'>101</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?001+ful+CHAP0996'>996</a>; 2004, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?041+ful+CHAP0794'>794</a>.</p>
