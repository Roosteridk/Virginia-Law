# § 59.1-284.01 Expiration of chapter; exceptions.

<p>A. All provisions of this chapter except §§ <a href='http://law.lis.virginia.gov/vacode/59.1-279/'>59.1-279</a>, <a href='http://law.lis.virginia.gov/vacode/59.1-280/'>59.1-280</a>, <a href='http://law.lis.virginia.gov/vacode/59.1-280.1/'>59.1-280.1</a>, <a href='http://law.lis.virginia.gov/vacode/59.1-282.1/'>59.1-282.1</a>, <a href='http://law.lis.virginia.gov/vacode/59.1-282.2/'>59.1-282.2</a> and this section shall expire on July 1, 2005, unless extended by an act of the General Assembly.</p><p>B. All enterprise zones designated pursuant to §§ <a href='http://law.lis.virginia.gov/vacode/59.1-274/'>59.1-274</a>, <a href='http://law.lis.virginia.gov/vacode/59.1-274.1/'>59.1-274.1</a>, and <a href='http://law.lis.virginia.gov/vacode/59.1-274.2/'>59.1-274.2</a> as those were in effect prior to July 1, 2005 shall continue in effect until the end of their 20-year designation period. Such zones shall be governed by the provisions of Chapter 49 (§ <a href='http://law.lis.virginia.gov/vacode/59.1-538/'>59.1-538</a> et seq.).</p><p>1995, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?951+ful+CHAP0792'>792</a>; 2005, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?051+ful+CHAP0863'>863</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?051+ful+CHAP0884'>884</a>.</p>
