# ยง 50-73.127 Effect of conversion; entity unchanged.

<p>A. A limited partnership that has been converted pursuant to this article is for all purposes the same entity that existed before the conversion.</p><p>B. When a conversion takes effect:</p><p>1. All property owned by the converting limited partnership remains vested in the resulting partnership;</p><p>2. All obligations of the converting limited partnership continue as obligations of the resulting partnership; and</p><p>3. An action or proceeding pending against the converting limited partnership may be continued as if the conversion had not occurred.</p><p>1996, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?961+ful+CHAP0292'>292</a>; 2007, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?071+ful+CHAP0631'>631</a>.</p>
