# ยง 50-73.142 Limited partnerships as registered limited liability partnerships.

<p>A domestic limited partnership may become a registered limited liability limited partnership by complying with the applicable provisions of the Virginia Revised Uniform Limited Partnership Act.</p><p>1996, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?961+ful+CHAP0292'>292</a>.</p>
