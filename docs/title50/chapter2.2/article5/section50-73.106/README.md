# ยง 50-73.106 Partner's transferable interest in partnership.

<p>The only transferable interest of a partner in the partnership is the partner's share of the profits and losses of the partnership and the partner's right to receive distributions. The interest is personal property.</p><p>1996, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?961+ful+CHAP0292'>292</a>.</p>
