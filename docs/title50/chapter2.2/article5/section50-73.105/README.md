# ยง 50-73.105 Partner not co-owner of partnership property.

<p>A partner is not a co-owner of partnership property and has no interest in partnership property which can be transferred, either voluntarily or involuntarily.</p><p>1996, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?961+ful+CHAP0292'>292</a>.</p>
