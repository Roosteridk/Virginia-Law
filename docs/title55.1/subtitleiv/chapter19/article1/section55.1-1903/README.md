# ยง 55.1-1903 Separate assessments, titles, and taxation.

<p>Except as otherwise provided in this section, each condominium unit constitutes a separate parcel of real estate. If there is any unit owner other than the declarant, each unit, together with its common element interest, but excluding its common element interest in convertible land and in any withdrawable land within which the declarant has the right to create units or limited common elements, shall be separately assessed and taxed. Each convertible land and withdrawable land within which the declarant has the right to create units or limited common elements shall be separately assessed and taxed against the declarant.</p><p>1974, c. 416, ยง 55-79.42; 1986, c. 324; 2019, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?191+ful+CHAP0712'>712</a>.</p>
