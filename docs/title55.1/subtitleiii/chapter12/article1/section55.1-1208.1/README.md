# ยง 55.1-1208.1 Rental agreements; child care.

<p>A rental agreement may contain provisions that allow the operation of child care services provided by a tenant of an apartment building that meet state and local laws and regulations.</p><p>2022, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?221+ful+CHAP0267'>267</a>.</p>
