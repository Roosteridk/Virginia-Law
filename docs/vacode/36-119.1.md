# ยง 36-119.1 Existing buildings.

This chapter shall not supersede provisions of the Fire Prevention Code promulgated by the Board under ยง [27-97](http://law.lis.virginia.gov/vacode/27-97/), that prescribe standards to be complied with in existing buildings or structures, provided that such regulations shall not impose requirements that are more restrictive than those of the Uniform Statewide Building Code under which the buildings or structures were constructed.
