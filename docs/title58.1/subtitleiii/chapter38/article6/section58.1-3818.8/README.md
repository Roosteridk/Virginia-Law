# § 58.1-3818.8 (Effective September 1, 2021) Definitions.

<p>As used in this article, unless the context requires a different meaning:</p><p>"Accommodations" means any room or space for which tax is imposed on the retail sale of the same pursuant to this article.</p><p>"Accommodations fee" means the same as such term is defined in § <a href='/vacode/58.1-602/'>58.1-602</a>.</p><p>"Accommodations intermediary" means the same as such term is defined in § <a href='/vacode/58.1-602/'>58.1-602</a>.</p><p>"Accommodations provider" means the same as such term is defined in § <a href='/vacode/58.1-602/'>58.1-602</a>.</p><p>"Affiliate" means the same as such term is defined in § <a href='/vacode/58.1-439.18/'>58.1-439.18</a>.</p><p>"Discount room charge" means the same as such term is defined in § <a href='/vacode/58.1-602/'>58.1-602</a>.</p><p>"Retail sale" means a sale to any person for any purpose other than for resale.</p><p>"Room charge" means the same as such term is defined in § <a href='/vacode/58.1-602/'>58.1-602</a>.</p><p>2021, Sp. Sess. I, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?212+ful+CHAP0383'>383</a>.</p>
