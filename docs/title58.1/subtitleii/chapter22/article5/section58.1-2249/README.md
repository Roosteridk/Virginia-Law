# ยง 58.1-2249 Tax on alternative fuel.

<p>There is hereby levied a tax at the rate levied on gasoline and gasohol on liquid alternative fuel used to operate a highway vehicle by means of a vehicle supply tank that stores fuel only for the purpose of supplying fuel to operate the vehicle. There is hereby levied a tax at a rate equivalent to that levied on gasoline and gasohol on all other alternative fuel used to operate a highway vehicle. The Commissioner shall determine the equivalent rate applicable to such other alternative fuels.</p><p>2000, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?001+ful+CHAP0729'>729</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?001+ful+CHAP0758'>758</a>; 2007, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?071+ful+CHAP0896'>896</a>; 2012, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?121+ful+CHAP0729'>729</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?121+ful+CHAP0733'>733</a>; 2013, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?131+ful+CHAP0766'>766</a>; 2014, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?141+ful+CHAP0014'>14</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?141+ful+CHAP0043'>43</a>; 2020, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?201+ful+CHAP1230'>1230</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?201+ful+CHAP1275'>1275</a>.</p>
