# § 58.1-815.4 (Contingent expiration dates) Distribution of recordation tax to the Commonwealth Transportation Fund.

<p>Of the state recordation taxes imposed pursuant to §§ <a href='/vacode/58.1-801/'>58.1-801</a> and <a href='/vacode/58.1-803/'>58.1-803</a>, the revenues collected each fiscal year from $0.03 of the total tax imposed under each section shall be deposited by the Comptroller into the Commonwealth Transportation Fund established pursuant to § <a href='/vacode/33.2-1524/'>33.2-1524</a>.</p><p>2007, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?071+ful+CHAP0896'>896</a>; 2013, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?131+ful+CHAP0639'>639</a>; 2015, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?151+ful+CHAP0684'>684</a>; 2018, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?181+ful+CHAP0854'>854</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?181+ful+CHAP0856'>856</a>; 2020, cc. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?201+ful+CHAP1230'>1230</a>, <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?201+ful+CHAP1275'>1275</a>.</p>
