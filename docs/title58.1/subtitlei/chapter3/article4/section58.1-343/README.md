# ยง 58.1-343 Place of filing.

<p>A. Every resident who is required by this chapter to file a return shall file his return with the commissioner of the revenue for the county or city in which he resides, and every nonresident who is required by this chapter to file a return shall file his return with the commissioner of the revenue for the county or city in which all or a part of his income from Virginia sources was derived.</p><p>B. Every fiduciary required to file a return on behalf of an individual shall file such return with the commissioner of the revenue having jurisdiction in the county or city in which the fiduciary qualified or, if there has been no qualification in this Commonwealth, in the county or city in which such fiduciary resides, does business or has an office or wherein the beneficiary or any of them may reside, or with the Department if provided by regulation thereof.</p><p>Code 1950, ยง 58-151.064; 1971, Ex. Sess., c. 171; 1984, c. 675.</p>
