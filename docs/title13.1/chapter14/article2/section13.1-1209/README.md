# ยง 13.1-1209 Purposes.

<p>Every business trust formed under this chapter has the purpose of engaging in any lawful business, except as otherwise may be provided by the law of this Commonwealth, unless a more limited purpose is set forth in the articles of trust.</p><p>2002, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?021+ful+CHAP0621'>621</a>.</p>
