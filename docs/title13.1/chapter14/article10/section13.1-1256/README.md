# § 13.1-1256 Collection by suit and of unpaid bills.

<p>The provisions of §§ <a href='http://law.lis.virginia.gov/vacode/13.1-775.1/'>13.1-775.1</a> and <a href='http://law.lis.virginia.gov/vacode/58.1-2814/'>58.1-2814</a>, so far as they are applicable, shall apply to the annual registration fees and penalties imposed by this chapter.</p><p>2002, c. <a href='http://lis.virginia.gov/cgi-bin/legp604.exe?021+ful+CHAP0621'>621</a>.</p>
